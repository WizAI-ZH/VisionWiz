## VisionWiz 1.3.0

### English
- Release version: v1.3.0
- Release date: 2026-05-07
- Manual update guide: https://vesibit.yuque.com/ednd8n/visionwiz/intro

### Highlights
- The update prompt now shows the full release notes instead of a truncated summary.
- Release notes in the updater now include bilingual English and Chinese content.
- The automatic update download window has been resized and reorganized to remove the ugly clipped layout and page scrollbar.
- Download progress now shows estimated time remaining in addition to transferred size and speed.
- If a download fails, you can try resuming the existing partial download first, then restart the download if needed.
