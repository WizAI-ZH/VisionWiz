## VisionWiz 1.4.3

### English

- Release version: `v1.4.3`
- Release date: `2026-05-11`
- Manual update guide: https://vesibit.yuque.com/ednd8n/visionwiz/intro

### Highlights

- Added selectable data augmentation modes for image classification and object detection training.
- Added class-level dataset statistics to training startup logs.
- Improved object detection dataset loading so unrelated or invalid content is skipped with warnings.

### 中文

- 发布版本：`v1.4.3`
- 发布日期：`2026-05-11`
- 手动更新说明：https://vesibit.yuque.com/ednd8n/visionwiz/intro

### 更新亮点

- 为图像分类和目标检测训练新增可选择的数据增强模式。
- 在训练启动日志中加入类别级数据量统计。
- 增强目标检测数据集读取鲁棒性，无关或无效内容会被跳过并记录警告。
