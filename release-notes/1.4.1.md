## VisionWiz 1.4.1

### English
- Release version: `v1.4.1`
- Release date: `2026-05-09`
- Manual update guide: https://vesibit.yuque.com/ednd8n/visionwiz/intro

### Highlights
- Improved how quickly physical serial disconnects are invalidated, so the authentication dialog returns sooner.
- Added higher-frequency presence monitoring for the authenticated serial port to reduce quick replug bypass cases.
- Cleared preview and authentication state immediately on serial close/error events to avoid stale sessions.

### 中文
- 发布版本：`v1.4.1`
- 发布时间：`2026-05-09`
- 手动更新说明：https://vesibit.yuque.com/ednd8n/visionwiz/intro

### 更新亮点
- 优化串口物理断开后的失效判定速度，让验证弹窗能更快重新出现。
- 为当前已认证串口增加更高频的存在性巡检，降低短时间断开后快速重连绕过重新验证的情况。
- 在串口 close/error 回调中立即清空预览与认证状态，避免旧连接状态残留。
