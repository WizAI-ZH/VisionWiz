## VisionWiz 1.4.6

### English

- Improved automatic updates so the installer starts only after VisionWiz has fully closed, reducing install failures caused by files still being in use.
- Update installers are now cached in a persistent application update directory instead of the system temp folder.
- Completed update downloads are reused, partial downloads can resume, and abnormal cached files are re-downloaded.
- The restart-download action still forces a clean download when users need to discard a bad cached installer.

### 中文

- 优化自动更新安装流程，安装程序会在 VisionWiz 完全关闭后再启动，减少程序文件仍被占用导致的安装失败。
- 自动更新安装包现在保存到持久化的应用更新缓存目录，不再依赖系统临时目录。
- 已完整下载的安装包会直接复用，未完成下载可以断点续传，异常缓存文件会重新下载。
- 保留“重新下载”操作，用户在需要时可以丢弃异常缓存并重新获取干净安装包。
