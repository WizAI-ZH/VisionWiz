## VisionWiz 1.5.2

### English

- Test release for validating automatic updates from VisionWiz 1.5.1.
- Keeps the PowerShell update helper, silent installation into the current VisionWiz directory, and automatic app restart behavior.
- No additional training workflow changes are included in this test release.
