## VisionWiz 1.4.5

### English

- Added search to the image classification and object detection training consoles. Use `Ctrl+F` or the right-click menu, then jump between matches with previous/next controls.
- Added a right-click clear-screen action for training consoles. It only clears the visible terminal output and does not stop training or delete saved logs.
- Improved training records with complete terminal transcript saving. Saved logs now remove terminal control codes and format progress updates as readable lines.
- Improved object detection XML compatibility for files without `size` or `difficult` fields while preserving standard Pascal VOC support.
- Added automatic retries for occasional NCC KModel conversion crashes.

### 中文

- 为图像分类和目标检测训练控制台新增搜索功能，可通过 `Ctrl+F` 或右键菜单打开，并支持上一个/下一个结果跳转。
- 为训练控制台新增右键清屏操作，只清空当前可见终端内容，不会停止训练，也不会删除已保存日志。
- 优化训练记录，新增完整终端输出保存；保存日志会清理终端控制码，并将训练进度整理成更易读的多行文本。
- 优化目标检测 XML 兼容性，支持缺少 `size` 或 `difficult` 字段的标注文件，同时保留标准 Pascal VOC 格式支持。
- 为 NCC KModel 转换阶段增加自动重试，降低转换器偶发崩溃导致训练失败的概率。
