## VisionWiz 1.4.7

### English

- Added search to historical training logs in image classification and object detection training records.
- Training record logs now support `Ctrl+F`, previous/next navigation, result counts, and highlighted matches.
- Reduced noisy progress output in saved and displayed training logs by keeping only the completed progress line for each epoch.
- Added the selected data augmentation mode to model information in both successful and failed training records.
- Added localized labels for the new data augmentation field in Simplified Chinese, Traditional Chinese, and English.

### 中文

- 为图像分类和目标检测的历史训练日志新增搜索功能。
- 训练记录日志支持 `Ctrl+F`、上一个/下一个、结果计数和命中高亮。
- 精简保存和展示的训练进度日志，每一轮只保留完成后的进度行，减少中间 batch 进度刷屏。
- 在成功和失败训练记录的模型信息中新增本次训练选择的数据增强模式。
- 为新的数据增强字段补充简体中文、繁体中文和英文界面文案。
