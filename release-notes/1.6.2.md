## English

- Improved image classification and object detection training consoles so backend terminal width matches the visible xterm console.
- Moved training-page side action buttons left to avoid overlapping the terminal scrollbar.
- Capture-record loading now skips unrelated folders, non-image files, unreadable images, and missing directories instead of crashing the main process.
- Added CVAT native XML annotation support for object detection training while preserving Pascal VOC, LabelImg, and relaxed VOC XML compatibility.

## 中文

- 优化图像分类和目标检测训练控制台，让后台终端宽度与前端 xterm 控制台可视宽度保持一致。
- 将训练页右侧悬浮操作按钮左移，避免与控制台滚动条重合。
- 图片拍摄记录读取现在会跳过无关文件夹、非图片文件、坏图片和不存在的目录，不再导致主进程崩溃。
- 新增目标检测训练对 CVAT 原生 XML 标注格式的支持，同时保留 Pascal VOC、LabelImg 和宽松 VOC XML 兼容。
