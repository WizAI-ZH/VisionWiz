## VisionWiz 1.5.1

### English

- Fixed the automatic update helper getting stuck as a `find "PID"` command window.
- Replaced the old `.cmd` update helper with a visible PowerShell helper that waits for VisionWiz using `Wait-Process`.
- Automatic updates still install silently into the current VisionWiz install directory.
- VisionWiz now restarts automatically after a successful automatic update.

### 中文

- 修复自动更新助手卡成 `find "PID"` 命令窗口的问题。
- 将旧的 `.cmd` 更新助手替换为可见的 PowerShell 更新助手，使用 `Wait-Process` 等待 VisionWiz 退出。
- 自动更新仍会静默安装到当前 VisionWiz 安装目录。
- 自动更新成功后会自动重新启动 VisionWiz。
