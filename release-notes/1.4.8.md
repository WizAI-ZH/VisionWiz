## VisionWiz 1.4.8

### English

- Improved automatic update installation with a visible update helper window.
- The helper waits for VisionWiz to close completely before installing the downloaded update.
- Automatic updates now install silently into the current VisionWiz installation directory, so users do not need to click through the installer wizard.
- If installation fails, the helper window stays open and shows the cached installer path for troubleshooting.

### 中文

- 优化自动更新安装流程，新增可见的更新助手窗口。
- 助手窗口会等待 VisionWiz 完全关闭后再安装已下载的更新。
- 自动更新会静默安装到当前 VisionWiz 所在目录，用户不需要逐步点击安装向导。
- 如果安装失败，助手窗口会保留并显示缓存安装包路径，方便排查。
