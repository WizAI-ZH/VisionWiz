## English

- Fixed the graphical automatic update helper not appearing after VisionWiz exits.
- The helper now starts independently through Windows and hides only the PowerShell host console while keeping the graphical progress window visible.
- Added foreground activation and helper startup logging for easier troubleshooting.
- Fixed `npm run update:test` by clearing `ELECTRON_RUN_AS_NODE` before launching Electron.

## 中文

- 修复 VisionWiz 退出后图形化自动更新助手不显示的问题。
- 更新助手现在通过 Windows 独立启动，并且只隐藏 PowerShell 宿主控制台，保留图形化进度窗口可见。
- 增加窗口置前和助手启动日志，方便排查更新流程问题。
- 修复 `npm run update:test`，启动 Electron 前会清除 `ELECTRON_RUN_AS_NODE`。
