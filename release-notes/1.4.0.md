## VisionWiz 1.4.0

### English
- Release version: v1.4.0
- Release date: 2026-05-08
- Manual update guide: https://vesibit.yuque.com/ednd8n/visionwiz/intro

### Highlights
- Added integrated K210 live preview in the data collection page after serial authentication.
- Improved capture workflows with burst capture, frame-based K210 saving, and a quick button to open capture folders.
- Fixed the serial authentication and preview IPC path so preview state changes propagate more reliably.
- Improved the startup sequence so the loading screen appears before the main and authentication windows.
- Normalized a batch of core files to UTF-8 to reduce Chinese text encoding problems.

### 中文
- 发布版本：v1.4.0
- 发布时间：2026-05-08
- 手动更新说明：https://vesibit.yuque.com/ednd8n/visionwiz/intro

### 更新亮点
- 图像采集页新增 K210 实时预览能力，认证成功后即可直接查看主控画面。
- 优化拍摄流程，新增连拍、按帧保存和拍摄记录一键打开路径按钮。
- 修复串口认证与预览相关的 IPC 通信问题，让预览状态同步更稳定。
- 优化启动顺序，优先显示加载页，再显示主窗口和认证窗口。
- 将一批核心文件统一为 UTF-8 编码，减少中文乱码问题。
