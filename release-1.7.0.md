# VisionWiz 1.7.0 Release Record

## 中文

- 版本号更新到 `1.7.0`。
- Make Sense 标注体验作为本次主要版本升级内容：复制上一张矩形框、右键按鼠标位置粘贴、撤销/重做、自然排序、多排序模式、导出后自动打开输出位置。
- 已同步更新 `README.md`、`CHANGELOG.md`、`release-notes/1.7.0.md` 的中英双语说明，Release 描述保持中文在前、英文在后。
- `dev:protectfull` 会先构建 Make Sense 三语言产物，再执行保护编译。

## English

- Version updated to `1.7.0`.
- This release focuses on Make Sense annotation improvements: previous-rectangle copy, right-click paste by mouse position, undo/redo, natural sorting, multiple sorting modes, and opening export locations automatically.
- `README.md`, `CHANGELOG.md`, and `release-notes/1.7.0.md` were updated with bilingual release notes, with Chinese first and English second.
- `dev:protectfull` builds all three Make Sense language variants before protected compilation.
